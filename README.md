# Toest
> 토익 리스닝 공부에 도움을 주는 앱

## 이런 기능이 있습니다!
* 업로드한 토익 음원의 스크립트 제공
* 한 번의 터치만으로 원하는 문장 듣기
* 문장 구간 반복 듣기
* 한 문장 반복 듣기

## 이런 기술들을 썼어요!
* MVP
* ButterKnife
* Realm DB
* Retrofit2
* MediaPlayer
* Lottie
* DeepSpeech
* FFMpeg
